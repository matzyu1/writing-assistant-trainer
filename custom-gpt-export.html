    Job Application Rules

    Main rulebook for CVs, cover letters and applications.

    These rules come before VoiceClone. VoiceClone can improve tone and clarity, but it must not override CV structure, cover letter formatting, ATS needs, file naming or factual accuracy.

    Build a prompt
    Job application rules override VoiceClone tone rules whenever there is a conflict.
    • If VoiceClone suggests a longer answer but CV rules require one page, keep the CV one page.
    • If VoiceClone suggests a natural paragraph but CV rules require bullet points, follow the CV rule.
    • If VoiceClone suggests extra detail but the documents do not support it, do not add it.
    • If VoiceClone suggests a softer tone but job rules require ATS keywords, keep truthful ATS keywords.

      Training Loop Instructions

      Repeat this loop until each document type is sufficient.

      1. Check which document types are lacking.
      2. Choose that type in Knowledge Trainer.
      3. Copy the generated prompt into your Custom GPT.
      4. Paste the Custom GPT output back into Knowledge Trainer.
      5. Paste your improved version and notes.
      6. Save and update the knowledge files.
      7. Manually upload the updated knowledge files to your Custom GPT.

      Minimum Useful Loop Target

      Aim for enough evidence, not endless tweaking.

      • LinkedIn posts: 3 to 5 feedback pairs
      • Interview answers: 3 to 5 feedback pairs
      • README: 2 to 3 feedback pairs
      • Emails: 2 to 3 feedback pairs

      You do not need to loop forever. Stop once the content type reaches Good enough, unless future outputs are still bad.
